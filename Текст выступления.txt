Здравствуйте. Сегодня я хотел бы представить вам свой проект по разработке приложения с графическим интерфейсом с помощью языка программирования python.

Мой проект посвящен десятипальцевому методу набора текста, который называется 
Слепая печать. Это особый метод набора текста на клавиатуре. На английском он называется touchtyping. Дословно - «печать на ощупь»‎. В документации моего проекта есть большой теоретический блок с информацией о печати вслепую.

Главное преимущество такого метода - скорость набора текста и удобство его применения, главный недостаток - сложности в обучении. Мой проект направлен как раз на то, чтобы эту сложность уменьшить и сделать обучение понятным и доступным.

Теперь немного расскажу о дизайне программы.

В верхней части окна приложения находятся два текстовых поля, одно из которых будет служить для ввода текста, а другое - для считывания. В левой части я расположил панель из 11 кнопок, по центру - интерактивную клавиатуру, которая в первое время значительно помогает в обучении слепой печати, также на одной панели с клавиатурой находятся кнопка для смены раскладки, кнопка для изменения цвета клавиатуры и секундомер.

Для того, чтобы приложение можно было считать готовым, нужно было реализовать ряд технических решений:

Сделать экранную клавиатуру интерактивной, то есть она должна показывать, какую клавишу пользователю нужно нажать зеленым цветом, а любую другую нажатую пользователем клавишу - серым цветом.

Дать пользователю возможность изменять стиль клавиатуры так, как ему захочется, то есть реализовать изменение цвета.

Сделать возможной работу в приложении не только на русском, но и на английском языке.

Дать пользователю возможность скрыть экранную клавиатуру.

Красным цветом обозначить неверный ввод текста пользователем.

Добавить в приложение режим практика, доступный при нажатии кнопки тренажер.

Добавить в приложение таймер для отсчета времени ввода текста.

Добавить в приложение режим обучение, доступный при нажатии одной из кнопок урок.

Программа получилась довольно большой - около 2000 строк кода. Все функции, перечисленные выше, были реализованы, приложение готово. Найти его можно в открытом доступе на GitHub . Ссылка на код программы есть в документации моего проекта.
